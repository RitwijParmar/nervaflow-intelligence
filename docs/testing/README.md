# NervaFlow Intelligence Testing Pack

This folder organizes the STLC assets for NervaFlow Intelligence, a supply-chain risk application with Django/DRF endpoints, Pub/Sub ingestion, graph-backed impact analysis, retrieval evidence, and fallback decision packets.

Use this index to move quickly:

- `stlc-plan.md` defines the testing lifecycle.
- `test-matrix.md` maps product behavior to test coverage.
- `traceability.md` links requirements to code and tests.
- `debugging-playbook.md` captures fast isolation steps for API, graph, and ingestion failures.
